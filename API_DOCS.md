
# 特征提取 API 文档

## 接口列表

### 1. 特征提取（JSON）
提取特征并以 JSON 序列化格式返回。

- **URL**：`/extract/json`
- **方法**：`POST`
- **Content-Type**：`multipart/form-data`

#### 请求参数

| 参数            | 类型   | 描述                                                                     | 必填 | 默认值                              |
| :-------------- | :----- | :----------------------------------------------------------------------- | :--- | :---------------------------------- |
| `video`         | 文件   | 要处理的视频文件二进制流（建议指定文件名和 MIME 类型，如 `video/mp4`）。 | 是   | -                                   |
| `config_path`   | 字符串 | 特征提取服务的配置文件在容器内的路径。                                   | 否   | `configs/mmsa_fet_av_language.yaml` |
| `skip_language` | 字符串 | 是否跳过语言特征提取（ASR）。可选值："true", "false"。                   | 否   | "false"                             |

#### 响应格式（JSON）

响应为包含提取特征信息的 JSON 对象。

```json
{
  "transcript": "这里是提取的字幕文本。",
  // 以下为特征的形状信息，仅作debug参考
  "audio_shape": [1, 925, 25],
  "video_shape": [1, 232, 177],
  "text_shape": [1, 3, 50],
  // 以上为特征的形状信息，仅作debug参考
  "raw_score": -0.20053596794605255 // 这里是情感得分，范围[-1, 1]
}
```
